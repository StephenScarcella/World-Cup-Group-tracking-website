How saving works
Everything you type here saves to your browser only (localStorage). It will not show up for other visitors. To publish:
  1. Make your edits (picks, avatars, chirps, etc.).
  2. Open Export / Import → click Download picks.json.
  3. Drag that file into your GitHub repo root (replace existing picks.json) → commit.
  4. Netlify rebuilds in ~30s. Now all 8 visitors load the same data. Refreshes won't wipe it.
Match scores are different — those auto-fetch throughout match days and save themselves. You only handle picks / groups / avatars / recap copy.
